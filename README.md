# QR Code Generator

A web application built with Node.js, Express, HTML, CSS, and JavaScript that generates customizable QR codes.

## Features

- Generate QR codes from any text
- Custom QR color
- Custom background color
- Input validation
- Clear button
- Responsive UI (Work in Progress)

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- QRCode Package
